# nlw-3
semana do next level week


## Requisitos

`node >= 12.0.0`

## Instalação

`npm install` ou `yarn install` 

## Iniciação do Front-end

abra a pasta `web` e use o comando:
`npm start` ou `yarn start`

## Iniciação do Back-end

abra a pasta `backend` e use o comando:
`npm dev` ou `yarn dev`
